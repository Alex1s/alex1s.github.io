Known Issues:

  • If the resolution is beeing converted, any mirroring or orientation conversion won' be applyed. (fixed in next version)
    Workaround:
    1. Convert only the resolution. Save the script.
    2. Use the saved script and convert the orientation.

  • General comment color doesnt update in the paste script screen if changed in settings. (fixed in next version)
    Workaround: Restart the app so that the change will take effect.

  • App crashes when pressing the "Interpet" if the script contains a touchDown command that selected a troop with a y coordinate >= 2040. (fixed in next version)
    Workaround: Try not to touch too close to the left and right edges of your screen.

  • The order of the troops in slot is dependent on what kind of order you selected them in the "Select Troops" Scene". (fixed in next version)
    Workaround: Select the troops in the correct order.
